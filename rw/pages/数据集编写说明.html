投稿方式

首先，请拿出诚意编写，不要随便投稿，内容不要与现有的重复太多！不可包含侵权或低创（低质量）mod/任务

方式：回帖上传附件，或发邮件至qzqz7@qq.com，json和图片一起打包。或在QQ频道内讨论

代码格式

可下载默认数据集并改写：../data/mission.json、../data/mod.json（右键另存为），json语法：JSON - JavaScript | MDN，推荐的编辑器：VS Code 或 Notepad3。

或者先编辑电子表格再转换为json：示例：【腾讯文档】示例数据集，你可以保存副本，再共享出去，非常适合多人合作。要注意转换为json后要批量修改一下links和tags。最后要手动加上"cid""name""desc""type"这些属性
